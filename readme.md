# Express + PostgreSQL + Prisma + Redis Learning Project

Project ini dibuat sebagai repo pembelajaran untuk memahami bagaimana membangun backend modern menggunakan **Express.js**, **PostgreSQL**, **Prisma ORM**, dan **Redis**.

Fokus utama repo ini bukan sekadar membuat API jalan, tapi memahami alur backend secara lebih utuh: mulai dari routing, controller, service, integrasi database relasional lewat ORM, sampai penggunaan Redis untuk kebutuhan cache atau optimisasi performa.

## Tujuan Project

Repo ini dibuat untuk belajar:

- membangun REST API dengan **Express**
- menghubungkan aplikasi ke **PostgreSQL**
- menggunakan **Prisma ORM** untuk query dan manajemen data
- memahami peran **Redis** dalam backend, terutama untuk caching
- membiasakan struktur kode backend yang lebih rapi dan modular
- melatih alur development backend dari local development sampai siap dideploy
